# 🗺️ Mapa do Crime – Projeto em Ciência de Dados

Este projeto faz parte da minha pesquisa acadêmica e tem como objetivo **analisar dados de criminalidade em regiões metropolitanas** através de um **pipeline ETL** e visualizações interativas no **Power BI**.  Projeto em fase de submissão acadêmica (FATEC).

---

O projeto combina **pipeline de dados (ETL)** e **dashboards interativos** para transformar dados brutos em informações úteis, apoiando pesquisas acadêmicas e oferecendo insights relevantes para a sociedade.

---

## 🎯 Objetivos
- Estruturar um **pipeline de dados** para coleta, tratamento e organização das informações.  
- Desenvolver **visualizações interativas** que permitam explorar padrões e tendências criminais.  
- Contribuir para estudos de **segurança pública e políticas sociais** com dados acessíveis e bem documentados.  

---

## 🛠️ Tecnologias
- **Python** → scripts de ETL e análise  
- **Pandas / NumPy** → manipulação e limpeza de dados  
- **SQL** → armazenamento e consultas  
- **Power BI** → dashboards e relatórios interativos  
- **GitHub** → versionamento e colaboração  

---

## 📂 Estrutura do Repositório
- `data/` → bases de dados brutas e tratadas  
- `etl/` → scripts de extração, transformação e carga  
- `dashboard/` → arquivos do Power BI  
- `docs/` → documentação e artigo científico  

---

## 📊 Dados Obtidos
A análise realizada entre **2017 e 2024** permitiu identificar:
- 📉 **Redução significativa** em determinados tipos de crimes em regiões metropolitanas específicas.  
- 📈 **Aumento de ocorrências** em áreas periféricas, especialmente crimes contra o patrimônio.  
- 🕒 **Picos sazonais** de criminalidade em determinados meses do ano.  
- 🌍 Diferenças marcantes entre **regiões metropolitanas**, evidenciando desigualdades sociais e econômicas.  
- 🔎 Dashboards interativos que permitem filtrar por ano, tipo de crime e região, facilitando a exploração dos dados.
